# DB Layer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create `crates/db` — a standalone SQLite database layer using sqlx, extracting all queries from the current D1-based `crates/api/src/handlers/`. Refactor `crates/shared` to remove D1 hacks.

**Architecture:** One new crate (`crates/db`) with modules per domain (lists, items, containers, tags, settings, preferences). All queries use `sqlx::query_as!` with compile-time checking. `crates/shared` types lose D1 deserializer hacks (`bool_from_number`, `u32_from_number`, `features_from_json`). Helpers (ownership checks, position, toggle) move to `db::helpers`.

**Tech Stack:** sqlx 0.8 (sqlite, runtime-tokio), uuid 1 (v4), serde, serde_json, tracing (instrument on all public db functions)

**Improvements over current code:**
- Single UPDATE per `items::update` call (dynamic query builder) instead of 5-10 round-trips
- Transactions where needed (create list + features, merge tags, reset list)
- `sqlx::Type` derive for `ContainerStatus` / `ListType` instead of manual serde string dance
- Feature validation (`check_item_features`) as pure function in shared, not coupled to HTTP layer
- SQL builder helpers for `by_date` / `calendar` UNION ALL variants (reduce copy-paste)
- `#[tracing::instrument(skip(pool))]` on all public db functions — automatic span logging with function name + args (skip pool/req to avoid noise)

**Reference:** Current queries live in `crates/api/src/handlers/{containers,items,lists,tags,settings,preferences}.rs` and `crates/api/src/helpers.rs`. Types in `crates/shared/src/lib.rs`.

### Task 3: Refactor shared types — remove D1 hacks

**Files:**
- Modify: `crates/shared/src/lib.rs`

The D1 quirks we remove:
- `bool_from_number` — sqlx maps `INTEGER` to `bool` natively
- `u32_from_number` — same, sqlx maps directly
- `features_from_json` — in db layer we'll handle the JSON aggregation differently

**Important:** The current `crates/api` still depends on these deserializers for D1. We do NOT break `crates/api`. Instead, we make the deserializers conditional or keep them but add clean alternatives.

Strategy: Add a `sqlx` feature flag to `kartoteka-shared`. When `sqlx` feature is enabled, structs derive `sqlx::FromRow`. When not, they keep the D1 deserializers. `crates/db` enables this feature; `crates/api` does not.

// crates/db/src/helpers.rs
use sqlx::SqlitePool;

/// Check if a resource belongs to the user.
pub async fn check_ownership(
    pool: &SqlitePool,
    table: &str,
    id: &str,
    user_id: &str,
) -> crate::Result<bool> {
    let sql = format!("SELECT id FROM {} WHERE id = ? AND user_id = ?", table);
    let row: Option<(String,)> = sqlx::query_as(&sql)
        .bind(id)
        .bind(user_id)
        .fetch_optional(pool)
        .await?;
    Ok(row.is_some())
}

/// Check if an item belongs to the user (via list join).
pub async fn check_item_ownership(
    pool: &SqlitePool,
    item_id: &str,
    user_id: &str,
) -> crate::Result<bool> {
    let row: Option<(String,)> = sqlx::query_as(
        "SELECT items.id FROM items \
         JOIN lists ON lists.id = items.list_id \
         WHERE items.id = ? AND lists.user_id = ?"
    )
    .bind(item_id)
    .bind(user_id)
    .fetch_optional(pool)
    .await?;
    Ok(row.is_some())
}

/// Check item ownership and return its list_id.
pub async fn check_item_ownership_with_list(
    pool: &SqlitePool,
    item_id: &str,
    user_id: &str,
) -> crate::Result<Option<String>> {
    let row: Option<(String,)> = sqlx::query_as(
        "SELECT items.list_id FROM items \
         JOIN lists ON lists.id = items.list_id \
         WHERE items.id = ? AND lists.user_id = ?"
    )
    .bind(item_id)
    .bind(user_id)
    .fetch_optional(pool)
    .await?;
    Ok(row.map(|r| r.0))
}

/// Toggle a boolean (INTEGER 0/1) field. Returns None if not found.
pub async fn toggle_bool_field(
    pool: &SqlitePool,
    table: &str,
    column: &str,
    id: &str,
    user_id: &str,
) -> crate::Result<Option<bool>> {
    let select_sql = format!("SELECT {} FROM {} WHERE id = ? AND user_id = ?", column, table);
    let row: Option<(bool,)> = sqlx::query_as(&select_sql)
        .bind(id)
        .bind(user_id)
        .fetch_optional(pool)
        .await?;

    let Some((current,)) = row else {
        return Ok(None);
    };

    let new_val = !current;
    let update_sql = format!(
        "UPDATE {} SET {} = ?, updated_at = datetime('now') WHERE id = ?",
        table, column
    );
    sqlx::query(&update_sql)
        .bind(new_val)
        .bind(id)
        .execute(pool)
        .await?;

    Ok(Some(new_val))
}

            )
            .bind(user_id)
            .fetch_one(pool)
            .await?
        }
    };

#[tracing::instrument(skip(pool, req))]
pub async fn create(
    pool: &SqlitePool,
    user_id: &str,
    req: &CreateContainerRequest,
) -> crate::Result<Container> {
    let id = uuid::Uuid::new_v4().to_string();

    // Validate parent is a folder (status IS NULL), not a project
    if let Some(ref parent_id) = req.parent_container_id {
        let parent_status: Option<(Option<String>,)> = sqlx::query_as(
            "SELECT status FROM containers WHERE id = ? AND user_id = ?"
        )
        .bind(parent_id)
        .bind(user_id)
        .fetch_optional(pool)
        .await?;
        match parent_status {
            None => return Err(crate::DbError::NotFound("container_not_found".into())),
            Some((Some(_),)) => return Err(crate::DbError::Validation("invalid_container_hierarchy".into())),
            Some((None,)) => {} // folder, OK
        }
    }

    let status_str: Option<String> = req.status.as_ref().map(|s| {
        serde_json::to_value(s).ok()
            .and_then(|v| v.as_str().map(String::from))
            .unwrap_or_else(|| "active".into())
    });

    let position = helpers::next_position_for_containers(pool, user_id, req.parent_container_id.as_deref()).await?;

    sqlx::query(
        "INSERT INTO containers (id, user_id, name, status, parent_container_id, position) \
         VALUES (?, ?, ?, ?, ?, ?)"
    )
    .bind(&id)
    .bind(user_id)
    .bind(&req.name)
    .bind(&status_str)
    .bind(&req.parent_container_id)
    .bind(position)
    .execute(pool)
    .await?;

    get_by_id(pool, &id).await
}

pub async fn get_one(
    pool: &SqlitePool,
    user_id: &str,
    id: &str,
) -> crate::Result<Option<ContainerDetail>> {
    // Track last opened
    let _ = sqlx::query(
        "UPDATE containers SET last_opened_at = datetime('now') WHERE id = ? AND user_id = ?"
    )
    .bind(id)
    .bind(user_id)
    .execute(pool)
    .await;

    let row: Option<ContainerRow> = sqlx::query_as(
        "SELECT id, user_id, name, description, status, parent_container_id, \
         position, pinned, last_opened_at, created_at, updated_at \
         FROM containers WHERE id = ? AND user_id = ?"
    )
    .bind(id)
    .bind(user_id)
    .fetch_optional(pool)
    .await?;

    let Some(row) = row else { return Ok(None) };
    let container: Container = row.into();

    // Compute progress
    let item_progress: (i32, i32) = sqlx::query_as(
        "SELECT \
         COALESCE(SUM(CASE WHEN i.completed = 1 THEN 1 ELSE 0 END), 0), \
         COUNT(i.id) \
         FROM items i JOIN lists l ON l.id = i.list_id \
         WHERE l.container_id = ?"
    )
    .bind(id)
    .fetch_one(pool)
    .await?;

    let list_progress: (i32, i32) = sqlx::query_as(
        "SELECT COUNT(*), \
         SUM(CASE WHEN NOT EXISTS (\
           SELECT 1 FROM items i2 WHERE i2.list_id = l.id AND i2.completed = 0\
         ) AND EXISTS (\
           SELECT 1 FROM items i3 WHERE i3.list_id = l.id\
         ) THEN 1 ELSE 0 END) \
         FROM lists l WHERE l.container_id = ?"
    )
    .bind(id)
    .fetch_one(pool)
    .await?;

    Ok(Some(ContainerDetail {
        container,
        completed_items: item_progress.0 as u32,
        total_items: item_progress.1 as u32,
        completed_lists: list_progress.1 as u32,
        total_lists: list_progress.0 as u32,
    }))
}

pub async fn update(
    pool: &SqlitePool,
    user_id: &str,
    id: &str,
    req: &UpdateContainerRequest,
) -> crate::Result<Container> {
    if !helpers::check_ownership(pool, "containers", id, user_id).await? {
        return Err(crate::DbError::NotFound("container_not_found".into()));
    }

    if let Some(name) = &req.name {
        sqlx::query("UPDATE containers SET name = ?, updated_at = datetime('now') WHERE id = ?")
            .bind(name).bind(id).execute(pool).await?;
    }
    if let Some(description) = &req.description {
        sqlx::query("UPDATE containers SET description = ?, updated_at = datetime('now') WHERE id = ?")
            .bind(description).bind(id).execute(pool).await?;
    }
    if let Some(status_opt) = &req.status {
        let status_str: Option<String> = status_opt.as_ref().map(|s| {
            serde_json::to_value(s).ok()
                .and_then(|v| v.as_str().map(String::from))
                .unwrap_or_else(|| "active".into())
        });
        sqlx::query("UPDATE containers SET status = ?, updated_at = datetime('now') WHERE id = ?")
            .bind(&status_str).bind(id).execute(pool).await?;
    }

    get_by_id(pool, id).await
}

pub async fn move_container(
    pool: &SqlitePool,
    user_id: &str,
    id: &str,
    req: &MoveContainerRequest,

    if !helpers::check_ownership(pool, "containers", id, user_id).await? {
        return Err(crate::DbError::NotFound("container_not_found".into()));
    }

    if let Some(ref parent_id) = req.parent_container_id {
        if parent_id == id {
            return Err(crate::DbError::Validation("invalid_container_move".into()));
        }
        let parent_status: Option<(Option<String>,)> = sqlx::query_as(
            "SELECT status FROM containers WHERE id = ? AND user_id = ?"
        )
        .bind(parent_id).bind(user_id)
        .fetch_optional(pool).await?;

            None => return Err(crate::DbError::NotFound("container_not_found".into())),
            Some((Some(_),)) => return Err(crate::DbError::Validation("invalid_container_hierarchy".into())),
            Some((None,)) => {}
        }
    }

    sqlx::query(
        "UPDATE containers SET parent_container_id = ?, updated_at = datetime('now') WHERE id = ?"
    )
    .bind(&req.parent_container_id)
    .bind(id)
    .execute(pool)
    .await?;

    get_by_id(pool, id).await
}

pub async fn put_locale(pool: &SqlitePool, user_id: &str, locale: &str) -> crate::Result<()> {

// crates/db/src/home.rs
use kartoteka_shared::db_rows::{ContainerRow, ListRow};
use kartoteka_shared::*;
use sqlx::SqlitePool;

use crate::containers::load_list_features;

pub struct HomeData {
    pub pinned_lists: Vec<List>,
    pub pinned_containers: Vec<Container>,
    pub recent_lists: Vec<List>,
    pub recent_containers: Vec<Container>,
    pub root_containers: Vec<Container>,
    pub root_lists: Vec<List>,
}

pub async fn get_home(pool: &SqlitePool, user_id: &str) -> crate::Result<HomeData> {
    let pinned_list_rows: Vec<ListRow> = sqlx::query_as(
        "SELECT id, user_id, name, description, list_type, parent_list_id, \
         position, archived, container_id, pinned, last_opened_at, created_at, updated_at \
         FROM lists WHERE user_id = ? AND pinned = 1 AND archived = 0 AND parent_list_id IS NULL \
         ORDER BY name ASC"
    ).bind(user_id).fetch_all(pool).await?;

    let pinned_container_rows: Vec<ContainerRow> = sqlx::query_as(
        "SELECT id, user_id, name, description, status, parent_container_id, \
         position, pinned, last_opened_at, created_at, updated_at \
         FROM containers WHERE user_id = ? AND pinned = 1 ORDER BY name ASC"
    ).bind(user_id).fetch_all(pool).await?;

    let recent_list_rows: Vec<ListRow> = sqlx::query_as(
        "SELECT id, user_id, name, description, list_type, parent_list_id, \
         position, archived, container_id, pinned, last_opened_at, created_at, updated_at \
         FROM lists WHERE user_id = ? AND pinned = 0 AND last_opened_at IS NOT NULL \
         AND archived = 0 AND parent_list_id IS NULL \
         ORDER BY last_opened_at DESC LIMIT 5"
    ).bind(user_id).fetch_all(pool).await?;

    let recent_container_rows: Vec<ContainerRow> = sqlx::query_as(
        "SELECT id, user_id, name, description, status, parent_container_id, \
         position, pinned, last_opened_at, created_at, updated_at \
         FROM containers WHERE user_id = ? AND pinned = 0 AND last_opened_at IS NOT NULL \
         ORDER BY last_opened_at DESC LIMIT 5"
    ).bind(user_id).fetch_all(pool).await?;

    let root_container_rows: Vec<ContainerRow> = sqlx::query_as(
        "SELECT id, user_id, name, description, status, parent_container_id, \
         position, pinned, last_opened_at, created_at, updated_at \
         FROM containers WHERE user_id = ? AND parent_container_id IS NULL \
         ORDER BY position ASC, created_at ASC"
    ).bind(user_id).fetch_all(pool).await?;

    let root_list_rows: Vec<ListRow> = sqlx::query_as(
        "SELECT id, user_id, name, description, list_type, parent_list_id, \
         position, archived, container_id, pinned, last_opened_at, created_at, updated_at \
         FROM lists WHERE user_id = ? AND container_id IS NULL AND parent_list_id IS NULL \
         AND archived = 0 ORDER BY updated_at DESC"
    ).bind(user_id).fetch_all(pool).await?;

    // Convert and load features for all lists
    async fn with_features(pool: &SqlitePool, rows: Vec<ListRow>) -> crate::Result<Vec<List>> {
        let mut lists: Vec<List> = rows.into_iter().map(Into::into).collect();
        for list in &mut lists {
            list.features = load_list_features(pool, &list.id).await?;
        }
        Ok(lists)
    }

    Ok(HomeData {
        pinned_lists: with_features(pool, pinned_list_rows).await?,
        pinned_containers: pinned_container_rows.into_iter().map(Into::into).collect(),
        recent_lists: with_features(pool, recent_list_rows).await?,
        recent_containers: recent_container_rows.into_iter().map(Into::into).collect(),
        root_containers: root_container_rows.into_iter().map(Into::into).collect(),
        root_lists: with_features(pool, root_list_rows).await?,
    })
}
